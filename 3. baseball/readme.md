# JavaScript를 활용한 야구게임

- 프로젝트 과정

1. 객체를 통한 야구게임 초기 설정과 정답 생성
2. 사용자의 입력값을 받아 예외처리 및 정답과 비교 후 출력
3. 게임 종료 후 결과 출려과 재시작 설정

<hr/>

이번 프로젝트를 통해 전체적인 로직을 함수를 이용하여 정리하였다.
js 배열의 특성을 활용하여 정답과 입력값을 비교하였다.
적절한 객체의 사용과 함수형 프로그래밍을 경험할 수 있었다.
