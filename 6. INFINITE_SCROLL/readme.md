# INFINITE SCROLL

자바스크립트를 이용한 무한 스크롤

- 프로젝트 과정

1. api 데이터 가져오기
2. 스크롤 이벤트 시 조건에 맞추어 특정 양의 데이터 가져오기
3. 모든 데이터 로드 시에 반복되는 api 요청 제어
4. 로딩 모션 제어

<hr>

스크롤 이벤트 시에 여러 요소의 height를 얻어와 다양한 방법으로 사용할 수 있었다.
불필요한 중복 요청을 제어할 수 있었다.
로딩 바를 통해 사용자에게 데이터 불러오는 과정을 시각적으로 보여줄 수 있었다.
